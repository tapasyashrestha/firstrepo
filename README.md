# Netflix Landing Page Clone

A modern and responsive Netflix-inspired landing page built using HTML and CSS. This project recreates the look and feel of Netflix’s homepage while focusing on clean UI design and responsive layouts.

---

## Features

* 🎬 Modern Netflix-style UI
* 📱 Fully Responsive Design
* 🖥️ Hero Section with Background Banner
* 📧 Email Signup Section
* 📺 Feature Showcase Sections
* ❓ FAQ Section
* 🎨 Hover Effects and Smooth Styling

---

## Tech Stack

* HTML5
* CSS3
